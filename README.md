# js

An assortment of graphics doodles, noodles and maybe other random stuff, implemented in vanilla JavaScript
(along with some minimal DOM manipulation -- programmatically creating the doodle-specific UI)

So far the .html are in a variety of hideous states, I'll be working on cleaning them up and standardizing my usage across doodles.
I'll also add additional discussion of each sample, though in general they should be usable and usage self-explanatory.

## Usage

Just point your browser to one of the .html files, it *should* "just work."  E.g. file:///path/to/js/arcs.html
